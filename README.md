### 🛠️ Technical Setup

1. **Environment Variables**: Create a `.env` file based on `.env.example`.
2. **Installation**: Run `npm install`.
3. **Migration**: To migrate existing HTML content to Notion, run `npm run migrate`.
4. **Development**:### Local Development

There are two ways to run the project locally:

1.  **`npm run dev`**: Standard fast development with HMR. Uses **Premium Mock Data** so you can style the UI without worrying about API limits or proxy setup.
2.  **`npm run dev:notion`**: Full stack development using Cloudflare Wrangler. This connects to your **Live Notion Database** using the proxy in `functions/api/notion.js`.

> [!TIP]
> Use `npm run dev` for 99% of UI work. Use `npm run dev:notion` only when you need to verify data mapping or live updates from Notion.
5. **Deployment**: Connect this repo to Cloudflare Pages. Add `VITE_NOTION_API_KEY` and `VITE_NOTION_DATABASE_ID` to your Cloudflare environment variables.
